.. _section-perform-installation:

Perform the installation
========================

Install the extension :ref:`the normal way in TYPO3
<t3coreapi:extension-install>`.

For `Composer <https://getcomposer.org/>`_ users:

.. code-block:: bash

   composer require aaw-team/pagenotfoundhandling

.. _section-get-source:

Get the source code
===================

.. code-block:: bash

   git clone https://github.com/aaw-team/pagenotfoundhandling.git
